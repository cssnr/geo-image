# Privacy Policy

This Privacy Policy applies to the GeoImage web extension (the "Application") for web browsers, developed by Shane (the "Service Provider") as an open-source project licensed under the [MIT License](LICENSE). By using the Application, you agree to the terms described in this Privacy Policy.

### Data Collection

The Service Provider does not collect, store, or retain any personal data. When you analyze an image, the Application transmits the data directly from your browser to the third-party API service you have configured, using the API key you provide. The Service Provider never receives, processes, or has access to any images or data.

### Data Storage

The Application does not store any data on external servers. Analysis results are stored on your browser's local storage and persists until you clear the data. The Application's settings, **including your API key**, are stored in your browser's web extension sync storage, which may sync across all browsers and devices linked to your browser's account. This synced data may be written to your device's file system in plain text.

### Third-Party Services

The Application transmits data directly to the third-party API service you have configured. The Service Provider has no control over how any third-party service receives, processes, stores, or uses that data. Please review the applicable policies below:

- [Google Privacy Policy](https://policies.google.com/privacy)
- [Google Gemini Privacy Notice](https://support.google.com/gemini/answer/13594961#privacy_notice)

This Privacy Policy may be updated from time to time. The Service Provider will update this page with any changes and revise the effective date below. Your continued use of the Application following the posting of changes constitutes your acknowledgment of those changes.
